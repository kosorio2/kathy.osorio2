    Forms

    Examples of Form Controls:

    • There is input: for fields, including text, passwords, radio buttons and file uploads
    • There is select: menus for drop down list of options
    • There is textarea: elements for longer text entry
    • There is button: elements for submitting and resetting forms

    Property Attributes and Descriptors

    • value - the value of the property and is undefined by default
    • writeable - shows whether a property can be changed or not, and is false by default
    • enumerable - shows whether a property will show up when the object is displayed in a for in loop, false by default
    • configurable - shows whether you can delete a property or change any of tis attributes, is false by default
