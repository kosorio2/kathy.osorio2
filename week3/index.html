    Math Examples:

    Coordinates of an Event:

    • The screenX and screenY properties show the number of pixels from the left and top of the screen where the event took place.
    • The clientX and clientY properties show the number of pixels from the left and top of the client that is being used
    • he pageX and pageY properties show the number of pixels from the left and top, respectively, where the event took place in the document

    Keyboard Events:

    • The keydown event occurs when a key is pressed and willcontinue to occurif the key is held down
    • The keypress event occurs after a keydown event but before a keyup event.
    • The keyup event occurs when a key is released.
